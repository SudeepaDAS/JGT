# 🚗 Tyre Shop Management System

A full-stack web application built with **React** (frontend) and **Node.js + Express + PostgreSQL** (backend) to manage tyre inventory, customers, suppliers, and sales efficiently.

---

## 🌟 Features

- 🧾 Tyre stock & brand management  
- 👥 Customer and supplier records  
- 🛒 Sales and purchase tracking  
- 📦 Inventory management  
- 📊 Dashboard with sales analytics  
- 🔐 Authentication & authorization (JWT)  
- ⚙️ RESTful API with Express.js  
- 🖥️ Modern, responsive React UI  

---

## 🧰 Tech Stack

**Frontend:**
- React.js  
- React Router  
- Axios  
- Bootstrap / TailwindCSS  

**Backend:**
- Node.js  
- Express.js  
- PostgreSQL  
- Sequelize ORM / Knex.js (for SQL queries)
